## 프로젝트 특징

* 프론트엔드와 백엔드를 분리하여 프로젝트 개발
    * 각 파트의 별도 Repository를 생성 후 작업
    * 프론트 : AWS S3 정적 호스팅
    * 백엔드 : AWS EC2 서버 호스팅
    * 빌드 후, S3와 EC2 연동
