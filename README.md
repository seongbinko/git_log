## 프로젝트 특징

* 프론트엔드와 백엔드를 분리하여 프로젝트 개발
    * 각 파트의 별도 Repository를 생성 후 작업
    * 프론트 : AWS S3 정적 호스팅
    * 백엔드 : AWS EC2 서버 호스팅
    * 빌드 후, S3와 EC2 연동

## 개요

* 명칭 : git_log
* 개발 인원 : 5명 (프론트 2명, 백엔드 3명)

* 주요 기능 : 회원가입 로그인(jwt), 전체 피드, 유저들의 상세피드, 피드 CRUD, 댓글, 좋아요 
* 개발 환경 : Springboot 2.4, Spring security, java 8, Mysql 8.0, Jpa, querydsl, Junit5
* 형상 관리 툴 : git
